#### Buscar información sobre los sistemas de ficheros en red más utilizados en la actualidad y comparar sus características. Hacer una lista de ventajas e inconvenientes de todos ellos, así como grandes sistemas en los que se utilicen. 
Aunque el más conocido es NFS, existen numerosos sistemas como:
-	AppleTalk
-	Samba (programa)
-	Server Message Block
-	Andrew File System
-	Secure Shell Filesystem
NFS va de la versión 1 a la 4, y es soportado por la mayoría de sistemas linux/UNIX, luego existen protocolos propietarios como algunos anteriores.
 
Los programas mas famosos que hacen uso de estos, dependiendo del sistema son:
-  Windows: Servicios para Network File System (NFS).- Integrado en el sistema, fácil de usar y configurar
-  Linux: nfs-kernel.- Bastante usado, fiable y libre
-  Linux: Samba- muy conocido y utilizado.
-  MacOS:NFS Share.- Incluido en las últimas versiones, compatible con la conexión entre sistemas.
-  MacOS: AppleTalk.- Ya no tan utilizado.
