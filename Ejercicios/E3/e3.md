#### Buscar con qué órdenes de terminal o herramientas gráficas podemos configurar bajo Windows y bajo Linux el enrutamiento del tráfico de un servidor para pasar el tráfico desde una subred a otra.
- Windows: En Windows existen varias utilidades como:
   - Comando route, se puede usar desde consola y permite definir rutas estáticas para distintas subredes
  - Configuración propia del sistema, asistente de configuración de red
- Linux: Como en Windows, Linux dispone de sus propias utilidades
  - Comando route, al igual que Windows, Linux dispone de este comando que permite el enrutamiento.
  - Propia configuración del sistema
  - Dnsmasq, como servicio DNS y dhcp permite la conexión de subredes

#### Buscar con qué órdenes de terminal o herramientas gráficas podemos configurar bajo Windows y bajo Linux el filtrado y bloqueo de paquetes.
- Linux:
  - netfilter
  - iptables
  - PF (Packet Filter)
  - Berkeley Packet Filter (BPF)
- Windows:
  - winapi
  - Windows Filtering Platform (WFP)
  - WinDivert
